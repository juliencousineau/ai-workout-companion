## Features Implemented

- ✅ Hevy API integration (auth, routines, workouts)
- ✅ AI chat interface with conversational guidance
- ✅ Rep-by-rep counting with motivational messages
- ✅ Timed exercise countdown support
- ✅ Workout summary on completion

## How to Use

1. Open `index.html` in browser
2. Enter Hevy API key (from hevy.com/settings → Developer)
3. Select a routine and start chatting!

> **Note:** Requires Hevy Pro subscription for API access.
